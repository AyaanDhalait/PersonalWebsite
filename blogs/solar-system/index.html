Simulating the Solar System in Python

I simulated the solar system in Python using multiple libraries. I started with Panda3D and Pygame for a prototype, and later moved to Ursina Engine with Pygame for the final version (v1.0). The result turned out better than I expected.

It didn’t just “work” — despite a lot of bugs — it was able to simulate stars, planets, comets, and even asteroids and meteoroids. These were spawned dynamically using Python’s random library.

Later, I expanded the idea and created an imaginary solar system. This project was originally intended for my school’s science fair, but I ran out of time and couldn’t complete it in time to present.

Even now, it still sits on my desktop, and I occasionally open it to experiment — especially with the collision system I had temporarily implemented (and later removed due to bugs).

Thanks for reading!
